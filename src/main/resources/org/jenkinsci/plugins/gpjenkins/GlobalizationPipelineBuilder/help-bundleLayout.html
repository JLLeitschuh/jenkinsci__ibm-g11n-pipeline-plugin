For Download goal

Specifies one of following keywords to control output file name or path in download goal.

  • LANGUAGE_SUFFIX: In the same directory with the source bundle file, with extra language suffix. For example, if the source bundle file is <Base Dir>com/ibm/g11n/MyMessages.properties, then the French version will be <Output Dir>com/ibm/g11n/MyMessages_fr.properties.
  • LANGUAGE_SUBDIR: In a language sub-directory under the directory where the source bundle file is placed. For example, if the source bundle file is <Base Dir>res/MyMessages.json, then the French version will be <Output Dir>res/fr/MyMessages.json.
  • LANGUAGE_DIR: In a language directory at the same level with the source bundle file. For example, if the source bundle file is <Base Dir>com/ibm/g11n/en/MyMessages.properties, then the French version will be <Output Dir>com/ibm/g11n/fr/MyMessages.properties.
  • LANGUAGE_ONLY: In the same directory with the source, using language code as the file name.For example, if the source file is <Base Dir>com/ibm/g11n/en.json,then the French version will be <Output Dir>com/ibm/g11n/fr.json. * Note: With this option, output file name will be just language code.(with file extension same with the source file) regardless of the source file name.
The default value is LANGUAGE_SUFFIX.
