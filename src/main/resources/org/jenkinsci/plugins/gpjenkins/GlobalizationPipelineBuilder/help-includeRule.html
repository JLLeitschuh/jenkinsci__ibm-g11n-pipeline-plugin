For Upload and Download goal

Specifies a set of source bundle files from Jenkins workspace.

Apart from setting Base Directory, You will also need to add Include and Exclude rules to fetch files. The inclusion/exclusion rules use the standard Ant-Style fileset syntax..
So, For example,
  • Base Directory : src/MyResources
  • Include Rule : **/*.json
  • Exclude Rule : **/config.json
will includes all *.json files under src/MyResources directory of your Jenkins workspace, but excludes all files with file name config.json .
  • Upload: Uploads source files in Base Directory filtred through Include and Exclude Rule to IBM GLobalization Pipeline instance.
  • Download: Downloads bundles from your instance of IBM Globalization Pipeline to Jenkins job workspace that matches the source files found from Base Directory filtered through Include and Exclude rule
